# Interactive Quiz Builder

A **React-based quiz builder** allowing users to create, edit, play, and review quizzes. All data is stored locally in the browser (`localStorage`).

## Features

- Create and edit quizzes with multiple questions and options.
- Play quizzes with real-time progress tracking.
- Review past quiz attempts with score and answers.
- Fully dark-themed UI.
